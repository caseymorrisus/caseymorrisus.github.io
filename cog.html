Cognitive Website

I was commissioned by Coginitive Gaming to create a responsive website for their professional gaming teams. The website was created as a custom WordPress theme to allow them to easily edit content. It included a featured stream on their homepage and other streams on the Streams page, all accessed via the Twitch API. Standard blog/news functionality was added. A custom system was created to allow their players to create guides for the suite of games in which they had professional teams.

Built with ♥ using Jade,Stylus,CoffeeScript,Gulp, and GitHub
